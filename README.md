🏥 Health Assistance System
A desktop Health Management System built with Java and JavaFX, backed by a MySQL database. It supports three user roles — Admin, Doctor, and Patient — each with their own dashboard and features.
Features

🔐 Secure login with role-based access (Admin / Doctor / Patient)
🛠️ Admin can manage users, appointments, medical records, and specialties
👨‍⚕️ Doctors can view patients and manage medical records
🧑‍💼 Patients can view appointments and medical history
🔔 Automated appointment reminders running in the background

Tech Stack:
Java & JavaFX (UI)
FXML (layouts)
MySQL + JDBC (database)
Eclipse IDE

Setup:
Clone the repo
Create a MySQL database named health_system
Update credentials in src/utils/DBConnection.java
Add JavaFX to your build path in Eclipse
Run Main.java
